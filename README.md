Hi, my name is Tanishq Singh Bisht, a fullstack developer and i have made this todo app using typescript and MERN stack

#Backend

##Authentication
1.User sign-up
2.User sign-in
3.User forgot/reset password

##Todo management
1. Create todo
2. Update todo
3. List todo
4. Delete todo
5. Mark todo as completed

#Frontend
1.React router for routing
2.Zod with react query for api data fetching
3.Zustand for global state management
4.React hook form for handling forms

###Additionally i have used resend to send emails to user for password reset request.
